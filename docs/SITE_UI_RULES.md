# サイトUIルール

## トップページ
- トップページ冒頭には、横長の特集記事カルーセルを配置する。
- カルーセルは4件の特集記事を自動で切り替え、ユーザーが前後ボタンとインジケーターで手動切り替えできるようにする。
- トップページには、サイトコンセプトの長い説明文を置かない。サイト説明はAboutページに集約する。
- カルーセル下には、カテゴリごとのトピックコーナーを順番に掲載する。
- 各カテゴリコーナーでは、そのカテゴリの記事を最大3件並べ、サムネイルは正方形で表示する。
- 記事数が3件未満のカテゴリでは、不足分にグレーの正方形プレースホルダーを表示する。
- 「よく読まれている記事」はトップページ下部に残す。
- 「目的から探す」ブロックはトップページには置かない。

## Aboutページ
- Aboutページには、サイトの目的、非公式メディアであること、公式素材を使わない方針を掲載する。
- HeaderとFooterからAboutページへ常設導線を置く。
- Aboutページから記事一覧とルール・マナー系記事へ移動できる導線を置く。
